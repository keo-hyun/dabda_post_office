# Dabda Post Office

Event web app for Dabda Post Office (2026).

## Scripts

- `npm run test:unit`
- `npm run test:e2e`

## Apps Script (clasp)

1. Install clasp: `npm i -g @google/clasp`
2. Login: `clasp login`
3. Replace `.clasp.json` `scriptId` with your Apps Script project ID
4. Push script files: `clasp push`
